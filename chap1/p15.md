```Scheme
(define (cube x) (* x x x))

(define (p x) (- (* 3 x) (* 4 (cube x))))

(define (sine angle)
  (if (not (> (abs angle) 0.1))
    angle
    (p (sine (/ angle 3.0)))))
```

# a
$12.15 / 3^n \le 0.1$ を満たすような最小の自然数 $n$ 回だけ `p` は作用させられる。

すなわち、$n = \lceil \log_3 (12.15 / 0.1) \rceil = 5$ である。

# b
`a` と同様にすれば、増加の程度はいずれも $\Theta(\log |a|)$ である。
